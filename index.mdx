title: "gsudo — sudo for Windows with UAC and shell support"
description: "gsudo brings the Unix sudo experience to Windows — elevate commands in your current console with UAC integration, credentials caching, and full shell support."
---

gsudo is a `sudo` equivalent for Windows that lets you run any command with elevated (admin) permissions directly in your current console window — no new window, no context-switching. Just prepend `gsudo` (or the built-in `sudo` alias) to any command and it runs elevated. Works with CMD, PowerShell, WSL, Git-Bash, and more.

## Get up and running

<Steps>
  <Step title="Install gsudo">
    Choose your preferred package manager:

    ```powershell
    winget install gerardog.gsudo
    # or
    choco install gsudo
    # or
    scoop install gsudo
    ```

    Restart your terminal after installation to refresh `PATH`.
  </Step>
  <Step title="Run your first elevated command">
    Prepend `gsudo` (or `sudo`) to any command:

    ```powershell
    gsudo notepad C:\Windows\System32\drivers\etc\hosts
    ```

    A UAC prompt will appear. After you approve, the command runs elevated in the same window.
  </Step>
  <Step title="Reduce UAC popups with cache">
    Enable Auto cache mode to behave like Unix sudo — one prompt, then elevated for 5 minutes:

    ```powershell
    gsudo config CacheMode Auto
    ```
  </Step>
  <Step title="Configure your shell (optional)">
    For PowerShell users, import the gsudo module for tab completion and `gsudo !!` support:

    ```powershell
    Write-Output "`nImport-Module `"gsudoModule`"" | Add-Content $PROFILE
    ```
  </Step>
</Steps>
